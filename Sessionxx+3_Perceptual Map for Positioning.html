Managing Customer Heterogeneity

Objectives of Perceptual Mapping

  • Consumers face an increasing number of choices in every category

  • If companies can find a way to stand out from the crowd, then they are likely to outperform their competitors.

  • Perceptual mapping can help with that

Objectives of Perceptual Mapping (cont.)

  • A perceptual map helps companies understand how consumers’ perceptions and preferences compare across different brands on key dimensions

  • What do we do with these understandings?

    • Differentiation: What dimension(s) do consumers consider as key as they compare and evaluate products and brands?

    • Positioning: What are the unique aspects of the brands? How are they different? What makes them stand out? Why should target customers purchase the focal company’s products and brands and not those of competitors?

Objectives of Perceptual Mapping (cont.)

  • In a perceptual map, each brand occupies a specific point.

  • The maps can be multi-dimensional

    • More than 2 or 3 dimensions are difficult to visualize

  • Brands that are more similar should show up on the map closer to each other relative to brands that are more dissimilar in the minds of consumers

  • These maps allow marketing managers to develop positioning strategies

  • Perceptual maps are generated using principal components analysis (PCA)

Uses of Perceptual Mapping in Marketing

  • To collect preference ratings, researchers usually ask respondents to provide preferences for brands.

  • Think about the supermarket chain market; you ask the following questions to shoppers

    “On a scale from 1 to 9, where 1 stands for poor and 9 for excellent, how would you rate Walmart/Aldi/Whole Foods/Wegmans/Costco on the following attributes: (1) Level of Prices; (2) Breadth of Assortment; (3) Quality of Service; (4) Level of Store Ambiance?”

  • Once the perception data is collected, principal components analysis (PCA) allows to represent them in a perceptual map

Principal components analysis (PCA)

  • PCA is often used to reduce the dimensionality of large data sets

    • Dimensionality reduction method

  • The goal of PCA is to explain most of the variability in the data with a smaller number of variables than the original data set

    • Suppose we have 100 bases and demographic variables

    • We could examine pairwise plots of each variable against every other variable

    • \(100*(100-1)/2 = 4,950\) combinations exist! -> not feasible

  • We would like to find a low-dimensional representation of the data without losing much information

  • PCA provides a tool to do just this.

Principal components analysis (cont.)

  • Suppose each \(n\) individual has \(p\) -dimension variables (e.g., age, gender, # purchases)

  • Not all of these variables are equally interesting

  • PCA seeks a small number of dimensions that are as interesting as possible

    • “Interesting” is measured by the amount of variation in the data

  • PCA is a linear combination of the \(p\) variables and reduces the number of plots necessary for visual analysis

Principal components analysis (cont.)

  • If we have \(p\) brand attributes \(x_1, x_2, ..., x_p\), the first principal component is the linear combination of the attributes:

\[ PC(1) = \phi_{(1)1}x_1 +\phi_{(1)2}x_2+... +\phi_{(1)p}x_p \]

  • \(\phi_{(1)j}\) principal component loading that contains the maximum variance of the data

  • After the first principal component (\(PC(1)\)) is determine, we can find the second principal component \(PC(2)\)

\[ PC(2) = \phi_{(2)1}x_1 +\phi_{(2)2}x_2+... +\phi_{(2)p}x_p \]

Principal components analysis (cont.)

  • The second principal component \(PC(2)\) is calculated similarly; it must be uncorrelated (orthogonal) with \(PC(1)\)

  • In PCA, principal components are extracted sequentially

    • The \(PC(1)\) accounts for the most variance in the data

  • We can similarly estimate \(PC(3)\), \(PC(4)\)…, but marketers prefer to retain only the first two components. Why?

  • The calculation of principal components requires linear algebra techniques (e.g., eigenvector)

    • You don’t have to know the details of math and statistics

  • Need to focus on the implementation and interpretation of the findings

PCA using R (cont.)

  • We see that there are 7 PCs
    • In general, there are \(min(n-1,p)\) informative PCs in a data set with n observations and p variables
  • Now, we can plot the first two principal components using biplot function
    • By default, the function uses the first two PCs
    • This is a rational default because the first two PCs contain the most variations in the data
    • If you want to (somehow) plot using principal component 3 and 4 then you can include option choices = 3:4 within biplot ; Not recommended!
biplot(pca_result)

PCA using R (cont.)

  • When comparing different attribute vectors, you can look at their relative positions to each other

  • The closer the two vectors, the more highly correlated

  • If the two vectors are pointing in the opposite direction, these vectors essentially mean the opposite (i.e., negatively correlated)

Takeaways

  • Consumers can choose from among many different goods and services in most markets

  • If companies can find a way to stand out in the competitive markets by appealing to their target segment, they stand much to gain

  • Perceptual maps through PCA allow managers to measure and represent customers’ perceptions of the brands

  • Once managers know customers’ perceptions and preferences, they can seek strategies to differentiate their brands vis-à-vis their competitors on key dimensions
